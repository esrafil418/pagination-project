## 🚀 Features

- ✅ Fetches todos from **JSONPlaceholder API**
- ✅ Displays data in a responsive **Bootstrap table**
- ✅ **Pagination** with Previous/Next buttons and page numbers
- ✅ Visual status indicators (Success/Pending with Bootstrap buttons)
- ✅ Loading state while fetching data
- ✅ Derived state for pagination (no extra state needed)
